La paleta de colores consta principalmente de una escala de grises. 
Esto no solo busca armonizar con la idea de que las fotos deban ser monocromaticas, tambien implica que el sitio es amigable con usuarios con discapacidades visuales tales como el daltonismo.
Se busca no saturar la vista de los usuarios con informacion innecesaria, por lo tanto se tomo un approach minimalista.
